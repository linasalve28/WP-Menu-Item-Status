== Description ==

This plugin lets you enable/disable menu items from menu.  So if you have a link in the menu that you want to disable for some time you can disable it. 

= Usage =

1. Go to Appearance > Menus
2. Edit the menu items accordingly
3. Change status either enable or disable and save
	if you choose enable, item will appear on menu in frontend.
	If you choose disable, item will not appear on menu in frontend.
	
== Installation ==

1. Upload the `plugin` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Appearance > Menus
4. Edit the menu items accordingly.
